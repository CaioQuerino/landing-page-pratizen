# Pratizen

Pratizen é uma landing page voltada para apresentar uma solução de comparação de preços e economia no cotidiano. A proposta da plataforma é ajudar consumidores a encontrar melhores ofertas, economizar tempo e tomar decisões de compra mais inteligentes.

## Objetivo

O projeto tem como objetivo divulgar a ideia do Pratizen de forma visual e objetiva, destacando:

- comparação de preços entre diferentes estabelecimentos;
- economia de tempo e dinheiro;
- listas inteligentes de compras;
- acesso a ofertas e encartes digitais;
- geolocalização para encontrar lojas próximas.

## Funcionalidades apresentadas

A landing page destaca as principais funcionalidades da proposta:

- Comparação de preços
- Listas inteligentes
- Geolocalização avançada
- Encartes digitais

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Como executar localmente

1. Clone este repositório:

```bash
git clone https://github.com/CaioQuerino/landing-page-pratizen.git
```

2. Acesse a pasta do projeto:

```bash
cd landing-page-pratizen
```

3. Abra o arquivo `index.html` em seu navegador.

Em seguida, acesse `http://localhost:5500`.
